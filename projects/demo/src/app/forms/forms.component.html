GTFormModule

How to start:

Import GTFormModule in the module you want to use

Components:

Generic editor:

Is a component that recives an object of type GTForm. With this object, it can generate a popup that shows a form with the specifications that you coded. For example, it can be use with an object called productForm:GTForm that specifies 4 attributes, each one with differente types, and you can open this form for creating, updating, and inspecting.

Example of code: 

    constructor(private GTFormService:GTFormService){}

    configForms(){
        this.gtFormProduct = new GTForm('create',['code','name','type','stock'],['Code','Full name','Type','Stock'],'Creating a product');
        this.gtFormProduct.addRequired(['name','type']);
        this.gtFormProduct.changeType('number',['stock']);
    }

    createProduct()
        this.gtFormProduct.changeTypeForm('create');
        this.GTFormService.showForm(this.gtFormProduct).subscribe(newProduct => {
            //Your api call for sending the new product
        });
    }

    editProduct(){
        this.gtFormProduct.changeTypeForm('update');
        this.GTFormService.showForm(this.gtFormProduct).subscribe(updatedProduct => {
            //Your api call for updating the product
        });
    }

    inspectProduct(){
        //It will disable the form
        this.gtFormProduct.changeTypeForm('inspect');
        this.GTFormService.showForm(this.gtFormProduct).subscribe();
    }
